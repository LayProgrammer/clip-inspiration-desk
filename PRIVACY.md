# 隐私说明

剪辑灵感台 Local Beta 是本地运行的软件。默认情况下，它不会把完整视频上传到本项目作者的服务器，因为本项目没有中心服务器。

## 本地保存什么

应用会在你的电脑上保存：

- 上传的视频素材：`storage/uploads/`
- 项目和灵感包数据：`storage/data.json`
- 缩略图和关键帧：`public/generated/`
- AI 配置：`.env.local`

这些路径都在项目目录内。

## 什么情况下会发送给 AI 服务商

如果你配置了智谱、OpenAI 或 OpenAI-compatible API key，应用会把以下内容发送给你配置的 AI 服务商：

- 少量关键帧图片
- 素材文件名
- 视频时长、分辨率、fps 等基础信息
- 你填写的创作目标
- 已生成的素材档案

应用不会主动把完整视频上传给 AI 服务商，但关键帧仍然可能包含人物、地点、物品等隐私信息。

## 请不要上传什么

请不要上传：

- 未经授权的人物肖像素材
- 身份证、银行卡、合同、病历等敏感信息
- 私密家庭视频
- 公司机密素材
- 任何你不希望第三方 AI 服务商看到的画面

## API key

API key 只保存在本机 `.env.local`。

请不要提交：

- `.env.local`
- `storage/`
- `public/generated/`

如果 key 曾经暴露，请立刻到服务商后台废弃旧 key 并重新生成。

## 清理数据

你可以在应用右上角“本地设置”里清理本地素材与项目数据。

清理会删除：

- `storage/`
- `public/generated/`

清理不会删除：

- `.env.local`
- 源代码
- README 或文档

## 开源作者声明

本项目作者无法看到你在本机运行时上传的素材或填写的 key。你使用哪个 AI 服务商，就需要遵守该服务商的隐私政策和服务条款。
